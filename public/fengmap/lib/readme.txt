fengmap.map.min.js		——地图核心模块，核心模块包括地图显示、覆盖物、事件、销毁、计算、地图工具等一些常用功能
fengmap.plugin.min.js	——地图插件模块，依赖于核心模块使用，包含地图 UI 控件、导航等功能
fengmap.analyser.min.js	——搜索分析模块，搜索查询、路径分析时引入；可独立于地图工作，支持Web Worker 和 Node
fengmap.effect.min.js	——特效渲染模块，依赖于核心模块使用，非特效渲染的项目不需要引入
toolBarStyle.css		——楼层控件样式文件，支持自定义样式
fengmap.plugins-track-player.min.js	——轨迹回放插件，依赖于核心模块使用
fengmap.plugins-mapedit.min.js	——编辑插件，包含框选及绘制/编辑点线面功能，依赖于核心模块使用
